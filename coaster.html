The Goal

The goal of this project is to model, 3D print, assemble, and program a functioning model roller coaster and corresponding HMIs (Human Machine Interfaces). This is a large project that I work on in my free time.

The Plan

The plan is to model a small existing coaster and build it with mainly 3D printed parts. I'd like this model to serve as a test bed that I can practice various control systems with. I plan to add multiple types of sensors to make it as realistic as possible.
